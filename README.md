# Jadsito Extension

### Descripción
Esta extensión ofrece una serie de snippets para facilitar el desarrollo de flujos en bots.

## Características

### Comandos Disponibles

| Comando     | Descripción                                                                                          |
|-------------|------------------------------------------------------------------------------------------------------|
| `fBasico`   | Snippet para crear un flujo básico con `addKeyword` y `addAnswer`.                                    |
| `eWelcome`  | Snippet para crear un flujo de bienvenida que se activa con el evento `EVENTS.WELCOME`.               |
| `eVoz`      | Snippet para crear un flujo que se activa al recibir una nota de voz con el evento `EVENTS.VOICE_NOTE`.|
| `eUbicacion`| Snippet para crear un flujo que se activa al recibir una ubicación con el evento `EVENTS.LOCATION`.    |
| `eVideoFoto`| Snippet para crear un flujo que se activa al recibir una foto, video o sticker con el evento `EVENTS.MEDIA`.|
| `eDocumento`| Snippet para crear un flujo que se activa al recibir cualquier tipo de documento con el evento `EVENTS.DOCUMENT`.|
| `ebMultimedia`| Snippet para crear un flujo con una respuesta que incluye multimedia (imagen, video, documento).    |
| `anDelay`   | Snippet para agregar un retraso en la respuesta.                                                      |
| `anCapture` | Snippet para capturar la respuesta del usuario.                                                       |
| `anCallback`| Snippet para crear un flujo con una función de callback asíncrona.                                    |
| `fFinalizar`| Snippet para crear un flujo con opción de cancelación y captura de datos.                             |
| `fGoTo`     | Snippet para redirigir a otro flujo.                                                                 |
| `ebVCARD`   | Snippet para enviar una tarjeta de contacto (VCARD).                                                  |
| `ebEmoji`   | Snippet para enviar una reacción con un emoji.                                                        |
| `ebResponder`| Snippet para responder citando un mensaje anterior.                                                  |
| `ebUbicacion`| Snippet para enviar una ubicación.                                                                   |
| `iMySQL`    | Snippet para configurar la conexión a una base de datos MySQL en un bot.                              |
